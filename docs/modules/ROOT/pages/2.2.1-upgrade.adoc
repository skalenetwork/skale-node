= {Network} {prev-version} -> {version} upgrade steps
include::parameters.adoc[]

[NOTE]
To address the issues occurred during testnet upgrade the procedure slightly differs from the one that is specified for the testnet.

== Run skale node turn-off

```shell
skale node turn-off --yes
```

== Stop docker-lvmpy service
```shell
systemctl stop docker-lvmpy
```

== Download node-cli binary

[source, subs="attributes"]
----
curl -L {node-cli-url} > /usr/local/bin/skale
----

== Verify checksum 

Verify checksum using sha512sum command

```shell
sha512sum /usr/local/bin/skale
```

Expected checksum:
[source, subs="attributes"]
----
{node-cli-checksum}
----

== Make node-cli executable

```shell
chmod +x /usr/local/bin/skale
```

== Modify .env

Update the following parameters to the new values.

[source, subs="attributes"]
----
CONTAINER_CONFIGS_STREAM={CONTAINER_CONFIGS_STREAM}
----

== Execute update

```shell
skale node update .env --yes
```
